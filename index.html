About This Page

  Why use Github User Page


GitHub User Page is public webpage hosted for free through GitHub. GitHub users can create and host both personal website and websites related to specific GitHub projects.


I wanted a personal website, simple and reliable, just to present my resume and maybe some of my blog posts, and apparently no need to make a database or third-party plugins. In this case, I selected Github User Page which is really lightweight and easy to be built from my Github repository.


I built this site from scratch, still trying to improve it in its layout and content. Welcome to visit my Github repository and pull request!
